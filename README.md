# 🧠 Segundo Cerebro

App web personal para organizar tu vida: finanzas, tareas, proyectos, hábitos y notas, todo conectado. Con soporte para parejas con espacio compartido.

## Stack

- **Frontend:** React + Vite
- **Backend:** Supabase (PostgreSQL + Auth + Realtime)
- **Deploy:** Vercel

## Setup rápido

1. Cloná el repo
2. Creá un proyecto en [Supabase](https://supabase.com)
3. Ejecutá `supabase/schema.sql` en el SQL Editor de Supabase
4. Copiá `.env.example` a `.env.local` y completá con tus claves
5. `npm install && npm run dev`

## Deploy en Vercel

1. Pusheá a GitHub
2. Importá en [Vercel](https://vercel.com)
3. Configurá las variables de entorno `VITE_SUPABASE_URL` y `VITE_SUPABASE_ANON_KEY`
4. Deploy!
